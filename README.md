# Jornada Viagens

Projeto desenvolvido durante o curso **HTML e CSS: Responsividade em Layouts**.

## 📌 Sobre o projeto

O objetivo foi construir o site da agência fictícia **Jornada Viagens**, garantindo uma experiência visual agradável e totalmente responsiva em diferentes dispositivos, como smartphones, tablets e desktops.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Git
- GitHub

## ✨ Funcionalidades

- Layout 100% responsivo
- Estrutura semântica em HTML
- Adaptação para diferentes tamanhos de tela
- Organização e versionamento do código com Git
- Otimização de imagens e meta tags para melhor performance e SEO
